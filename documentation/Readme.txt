Αυτό είναι η δεύτερη προσπάθεια δημιουργίας ενός style file για LaTeX 
για τις πτυχιακές εργασίες του ΤΕΙ Σερρών (Τμήμα Πληροφορικής & Επικοινωνιών).

Αρχεία, έκδοση 0.3
-------------------
icdthesisUTF.sty - Το style file της πτυχιακής
testUTF.tex - Παράδειγμα πτυχιακής με βάση το icdthesisUTF.sty
testUTF.bib - Παράδειγμα βιβλιογραφίας
lion.jpg - Εικόνα που χρησιμοποιείται από το test.tex
testUTF.pdf - το output από το XeLaTeX
readme.txt - Αυτό το αρχείο

Όλες οι δοκιμές έγιναν με xelatex και με τη διανομή TeXLive 2010 (for Mac)

Προσοχή:
--------
Για την παραγωγή της πολυ-γλωσσικής βιβλιογραφίας σε Unicode χρησιμοποιήθηκε το πρόγραμμα biblatex με το backend biber, αντί του BibTeX
